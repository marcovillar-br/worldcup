# archive/ — recordação da campanha 2026

**Fotografia final da Copa 2026, versionada de propósito.** Congelada em 2026-07-20, com os 104
jogos disputados (Espanha campeã; J104 `0×0` nos 90' + gol na prorrogação).

| Arquivo | O que é |
|---|---|
| `palpites-2026.md` | Tabela completa dos 104 jogos (formato canônico, diffável) |
| `palpites-2026.html` | Mesma tabela, autocontida e print-friendly |
| `palpites-2026.pdf` | Exportação manual do HTML (Ctrl+P) — nenhum script gera este arquivo |
| `apresentacao.html` | Deck do projeto no estado de fechamento da campanha |

⚠️ **Não é artefato de processo — é acervo.** Ao contrário de `out/` (regenerável, gitignored) e de
`history/` (snapshots diários que alimentam a medição de eficiência), nada aqui é lido por código:
é uma cópia parada, guardada por valor de recordação. **Não regenere, não "atualize" e não apague**
em limpeza de artefatos; se o modelo mudar, estes arquivos continuam retratando o fim da Copa 2026
como ele foi — que é justamente a graça.

Versionar HTML/PDF é uma **exceção deliberada** à convenção de não versionar gerados
(ver `AGENTS.md` → *Gerados*), decidida pelo usuário em 2026-07-20. Os arquivos contêm apenas
probabilidades **derivadas** do modelo/blend — nunca odds do feed licenciado, cuja redistribuição o
ToS da The Odds API proíbe (`docs/DATA.md` §6).
